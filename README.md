# Проект онлайн-обучения

Этот проект создан для онлайн-обучения с использованием Django и других технологий.

## Запуск проекта с помощью Docker Compose

Для запуска проекта с помощью Docker Compose выполните следующие шаги:

1. Установите Docker и Docker Compose, если они еще не установлены на вашем компьютере.

2. Склонируйте репозиторий с проектом:
    ```
    git clone https://github.com/AlekseiKostromin/homeworks_drf_mylms
    ```
3. Перейдите в директорию с проектом:
    ```
    cd my_lms_project
    ```
4. Создайте файл `.env` в корне проекта и укажите в нем переменные окружения:
    ```
    пример переменных окружения лежит в файле .env_sample
    ```
5. Соберите и запустите контейнеры Docker:
    ```
    docker-compose up --build
    ```
8. Откройте браузер и перейдите по адресу `http://localhost:8000` для доступа к  проекту.

9. При необходимости выполните миграции базы данных и соберите статические файлы:
    ```
    docker-compose exec web python manage.py migrate
    docker-compose exec web python manage.py collectstatic
    ```
10. Готово! Проект онлайн-обучения теперь запущен с помощью Docker Compose.
